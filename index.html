Aútomatas Celulares

Un modelo matemático para la simulación de comportamientos emergentes


Un sistema autómata celular es un modelo matemático abstracto, consistente en un conjunto de celdas, organizadas generalmente en forma de arreglo de n-dimensiones. Cada celda tiene dos o más estados posibles, y estos cambian en función del tiempo, en intervalos discretos, de acuerdo a un conjunto de reglas predeterminadas utilizando como entrada el estado local actual. En esta aplicación web podremos ejecutar tres tipos de simulaciones: Unidimensionales, Bidimensionales, y el Modelo Enjambres.

En este modelo, los datos de entrada para cada transición son el nodo actual y sus dos vecinos. Para cada combinación posible definimos un estado de salida: ocupado o vacío. Podemos asignar un nombre a una regla codificando cada salida como un dígito de un número binario.

En este modelo, cada celda está rodeada por ocho casillas. Podemos definir tos tipos de vecindarios: El vecindario Von Neumann está compuesto por únicamente los cuatrovecinos cardinales, mientras que el vecindario Moore está compuesto por esos cuatro además de los vecinos diagonales, para un total de ocho.

Una vez definido el vecindario, deberemos seleccionar que cantidades de vecinos resultarán en una entrada válida para que una ocupada lo continue estando o una celda vacía pase a estar ocupada.

En este modelo, en lugar de tener simplemente dos estados, vacío u ocupado, definimos cuatro grupos de enjambres que compiten por recursos.

Cada celda tendrá distintas puntuaciones de ciertos "recursos" que serán utilizados para definir el estado siguiente de cada celda y de sus vecinos.

Hemos definido tres recursos: Comida, Guerra y Cultura.

Para aspirar a capturar una casilla vacía, cada enjambre deberá reunir un mínimo de alimento. Si dos o más enjambres cumplen ese requisito, el que posea mayor puntuación de guerra ocupará de la celda. Si hay un empate, seguirá vacía.

Para tomar o mantener el control de una celda ocupada, nuevamente solo aquellos enjambres que reúnan el mínimo de alimento serán candidatos. En caso de haber dos o más, aquel grupo con mayor puntuación de cultura tomará control de la casilla. En caso de un empate, pasará a estar vacía.
